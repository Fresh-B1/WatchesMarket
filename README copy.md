# Описание

- Это дефолтная сборка окружения, делай npm i и работай

- При запуске (npm run dev) сразу будет "белая" main page

- Фалы пустых папках есть фафлы .gitkeep их можно удалять, они нужны чтобы пустые папки не удалялись в гите

- Имя БД можно поменять в соответсвующем файле db/config или просто замени файлы db на свою БД

- Преднаписаны скрипты в package.json:

  - для запуска nodemon,

  - отката и наката миграций,

  - для дропа базы и наката обратно.
